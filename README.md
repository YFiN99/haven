# Haven Exchange 

Haven is a high-performance decentralized exchange (DEX) and yield farming protocol built on the **DataHaven Testnet**. It enables users to swap assets, provide liquidity, and earn rewards through a sustainable revenue-sharing model.

##  Features

- **Decentralized Swap**: Low-latency token exchange using an Automated Market Maker (AMM) model.
- **Liquidity Provision**: Earn trading fees by providing liquidity to HAV/USDC and other pairs.
- **Yield Farming (Revenue Share)**: Stake your LP tokens in the HavenFarm to earn **$DATA** tokens distributed every second.
- **Real-time Rewards**: High-precision reward calculation (~0.00012 DATA per 30 seconds).
